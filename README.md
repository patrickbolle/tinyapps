# Tiny Apps

A collection of small, useful web applications built with vanilla JavaScript and modern web technologies.

## Apps

### 3x30 Workout Tracker

A Progressive Web App (PWA) for tracking Jake Wood's 3x30 Bodyweight Program.

**Live Demo:** [https://3x30-workout.pages.dev](https://3x30-workout.pages.dev)

**Features:**
- Full workout program with 3 phases (Movement Mastery, Volume Build, Density & Power)
- Smart timers with audio cues for warmup, rest periods, EMOM, AMRAP, and Tabata
- Exercise progression (Beginner, Intermediate, Advanced)
- Workout history and tracking
- Offline support (PWA)
- iPhone optimized

**Tech Stack:**
- Vanilla JavaScript
- Tailwind CSS
- localStorage API
- Service Worker
- Web Audio API
