# Tiny Apps

A collection of small, useful web applications built with vanilla JavaScript and modern web technologies.

## Apps

### Bodyweight Workout App

A Progressive Web App (PWA) for tracking Jake Wood's 3x30 Bodyweight Program.

**Live Demo:** [https://3x30-workout.pages.dev](https://3x30-workout.pages.dev)

**Features:**
- Full workout program with 3 phases (Movement Mastery, Volume Build, Density & Power)
- Smart timers with audio cues for warmup, rest periods, EMOM, AMRAP, and Tabata
- Exercise progression (Beginner, Intermediate, Advanced)
- Workout history and tracking
- Offline support (PWA)
- iPhone optimized

**Tech Stack:**
- Vanilla JavaScript
- Tailwind CSS
- localStorage API
- Service Worker
- Web Audio API

[View Source](./3x30-workout)

### Breathe App

A guided Progressive Web App (PWA) for practicing the Wim Hof breathing method.

**Live Demo:** [https://wim-hof-breathing.pages.dev](https://wim-hof-breathing.pages.dev)

**Features:**
- Guided breathing sessions with visual and audio cues
- Customizable sessions (breaths per round, number of rounds, pace)
- Animated breathing circle that expands/contracts with your breath
- Progress tracking with retention times and statistics
- Audio feedback for inhale, exhale, retention, and transitions
- Full Wim Hof method (deep breathing, retention, recovery breath)
- Offline support (PWA)
- iPhone optimized

**Tech Stack:**
- Vanilla JavaScript
- Tailwind CSS
- Web Audio API
- localStorage API
- Service Worker

[View Source](./wim-hof-breathing)

## Adding New Apps

Each app should:
- Be self-contained in its own directory
- Include a README with setup instructions
- Use modern web standards
- Be deployable to Cloudflare Pages or similar platforms
